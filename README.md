# SOC Log Analyzer

This project simulates a basic SOC analyst task using Python.

## Description
The script analyzes authentication logs to:
- Count failed and successful login attempts
- Detect possible brute-force attacks
- Generate a basic security alert

## Technologies Used
- Python
- Log Analysis

## SOC Relevance
This project demonstrates entry-level SOC analyst skills such as:
- Log monitoring
- Alert detection
- Security event analysis
